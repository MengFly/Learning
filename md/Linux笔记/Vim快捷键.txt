编辑Vimrc文件 :edit $VIM/_vimrc

:sav fileName  	: 保存文件，filename文件名
e		: 移动到单词的末尾位置
b		: 移动到单词的起始位置
0		：移动到一行的起始位置
gg		：移动到文件的起始位置
G		: 移动到文件的最后位置
number + G	: 移动到指定
G		: 移动到文件的最后位置
number + G	: 移动到指定行行
:number		：移动到文件的指定行
[[		：移动到函数的起始位置
{{		：移动到代码块的起始位置
2w		: 移动之后两个单词的起始处
3e		: 移动之后三个单词的末尾处
u		: 撤销
U		: 撤销对整行的修改
:w fileName	: 保存当前文件的内容到filename文件中
o		: 打开新的一行并进入编辑模式
O		: 在当前行上方打开一行并进入编辑模式
I		: 在当前行的起始位置开始编辑
A		: 在当前行的终止位置开始编辑
R		: 从指针位置开始进行替换





复制，剪贴，剪切
y		：复制选择的文字到剪贴板
v		: 进入可视化模式,用于选取内容
p		：粘贴
yy		: 复制当前一行
dd		：删除当前一行
2dd		: 从当前行开始删除两行
dw		: 删除到当前单词的下一个单词的起始处
de		: 删除到当前单词的最后
d$		: 从鼠标位置删除到一行的末尾(命令模式)
d^		: 从鼠标位置删除到一行的起始位置(命令模式)
d2w		: 删除两个单词
使用dd将一行删除后,再使用p命令可以把删除的一行粘贴到下一行


搜索
/word		：寻找单词（支持正则表达式）
Ctrl + G 	: 显示当前文件信息以及光标在文件中的位置
替换

:$s/old/new 		:替换整个文件中的字符
:$s/old/new/gc
:替换整个文件中的字符,并且在替换的时候会提示是否进行替换
:s/old/new		:在当前行替换第一个单词
:s/old/new/g		:在当前行替换所有单词
:2,35s/old/new/gi	:替换2-35行的单词(gi是忽略大小写)
:5,s/old/new		:替换第五行到最后的单词
Ctrl + X 		:让光标所在的位置的数字递减（命令模式）
r + newchar		:替换光标处的字符
cw			:删除到当前单词末尾,并进行编辑模式
c$			:删除到一行的末尾,并进行编辑模式
n 			:查找下一个
N			:查找上一个
%			:查找匹配的括号

大小写
Vu			：当前行小写（命令模式）
VU			：当前行大写（命令模式）
g~~			:Invert case（命令模式）

读写文件
:1,10 w outfile		: 保存当前文件的1-10行到outfile
:1,10 w >> outfile 	: 增加当前文件的1-10行到outfile
:r infile		: 在当前文件中插入infile中的内容
:23r infile		: 在当前文件的23行插入infile中的内容
:r !command		: 输入command执行的结果到当前文件
文件浏览器
:e 			: 打开文件
:Sex			: 打开文件管理器选择打开的文件（上下方式）
:Sex!			: 打开文件管理器选择打开的文件（左右方式）
:browse e		: 打开系统文件管理器去选择打开文件
:cd..			: 当前文件的路径移动到父路径
:args *.md		: 打开当前路径下面所有md结尾的文件
gf			: 打开光标所在位置的单词的文件（命令模式）

Tab
:tabnew			: 创建一个新的Tab
gt			：显示下一个tab（命令模式）
:tabfirst		: 显示第一个tab
:tablast		: 显示最后一个tab

缩进
>> 			: 缩进
<< 			: 反向缩进
:!command		: 执行cmd命令
