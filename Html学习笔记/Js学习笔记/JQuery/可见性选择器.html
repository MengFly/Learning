选择器 描述
$(":visible") 选择所有显示的元素
$(":hidden") 选择所有的隐藏元素

:hidden选择器，不仅仅包含样式是display="none"的元素，还包括隐藏表单，visibility等等

隐藏元素的几种方式

  1. CSS display的值为none
  2. type="hidden"的表单元素
  3. 高度和宽度都显示设置为0
  4. 一个祖先元素是隐藏的
  5. CSS visibility的值是hidden
  6. CSS opacity的值是0
    7. 元素的visibility:hidden或opacity:0被认为是可见的，因为他们仍然占用空间布局
